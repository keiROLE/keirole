GitHub520 是一个自动维护的 GitHub hosts 项目，解决国内访问 GitHub 速度慢的问题。

- **无需安装**：只需配置 hosts 文件即可生效
- **自动更新**：社区持续维护最新的 hosts 规则
- **效果显著**：大幅提升 GitHub 页面加载速度和图片显示
- **覆盖全面**：解决代码仓库、Issues、头像加载等全链路问题
